-----------------------------------------------------------------
This module Search and replace a string in all 
custom added block body and menus title.

Installation
-------------
* Copy the whole "search_replace_blocks_menus" directory to your modules
  directory - "sites/all/modules"
* Enable the module.
* Set your desired permissions at - "admin/people/permissions".

Usage
------   
 * Go to "/admin/config/system/replace-blocks-menus".
 * Fill required fields in form.

Permissions available
---------------------
* Access replace Blocks Menus Form
